# Olympic Medal Predictor

The Olympic Games are the world's only truly global, multi-sport, celebratory athletics competition. With more than 200 countries participating in over 400 events across the Summer and Winter Games, the Olympics are where the world comes to compete, feel inspired, and be together. Lets predict which country wins the most medals..A machine learning project that predicts the number of Olympic medals a country is expected to win based on historical Olympic performance and related features.

## Overview

The Olympic Medal Predictor uses historical Olympic data to build a predictive model capable of estimating medal counts for participating countries. The project demonstrates the complete machine learning workflow—from data collection and preprocessing to model training, evaluation, and prediction.

## Features

* Historical Olympic data analysis
* Data cleaning and preprocessing
* Feature engineering
* Machine Learning model training
* Medal count prediction
* Model performance evaluation
* Easy-to-use prediction workflow

* Python
* Pandas

* Scikit-learn
* Matplotlib

## Dataset

This project uses historical Olympic data containing country-wise medal counts and related information.

Example features include:

* Country
* Year
* Number of Athletes
* Previous Medal Count
* Gold Medals
* Silver Medals
* Bronze Medals

## Installation

Clone the repository:

```
git clone https://github.com/deveshvaidyaofficial/Olympic_Medal_Predictor.git
```

Navigate to the project directory:

```
cd Olympic_Medal_Predictor
```

Install dependencies:

```
pip install -r requirements.txt
```

## Usage

Run the Jupyter Notebook:

```bash
jupyter notebook
```

Open the project notebook and execute the cells in order to:

1. Load the dataset
2. Preprocess the data
3. Train the model
4. Evaluate performance
5. Generate medal predictions
